Debian Quick Image Baker pre-baked images

Here you can download pre-baked images created by DQIB. Each downloadable file is a ZIP archive containing the root filesystem, kernel, initrd and a readme text file (which includes launch command and auth info). Have fun and report bugs to the GitLab project.

For more information see DQIB's README or this blog post.

Any help on how to make work the architectures that currently don't work is greatly welcome!

Unsupported ports

  • arc and ia64 are not supported by QEMU, so there is no way to have it working
  • No work has been done for loong64 yet, but in theory it should work
  • x32 is not an actual architecture, but a different data model and ABI on top of amd64 CPUs; therefore it does not really fit DQIB's model, although at some point we could provide an x32 chroot
  • non-Linux ports (of which hurd-i386 is currently the only survivor) are not supported because I do not know how to bootstrap them from a Linux machine
